## Build Setup

``` bash
# 我是使用npm来安装依赖包和启动项目，没有用yarn
# install dependencies
npm install

# serve with hot reload at localhost:4000
npm run dev

# build with hot reload
npm run dev-proxy

# deploy
npm run deploy
```
